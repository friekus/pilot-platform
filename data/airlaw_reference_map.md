**Note on page numbers:** the "Page" column is the **compilation page number** printed in the footer of each PDF page, which is what a reviewer will see on the page (not the sequential PDF page index). Every CASR section heading below has been literally copied from the Act text.

**Scope-limited sources:**
- CASR Part 1 (Dictionary) and CASR Parts 42/43 are NOT in Vol 2. The CASR Dictionary is cited only for terms also reproduced verbatim in VFRG or MOS. Part 43 (maintenance release for class B aircraft) is referenced indirectly via CASR 91.145 / 91.110, which are primary sources.
- AIP (`complete_19MAR2026.pdf`) was listed in the prompt but is not present in `docs/`. Accordingly, questions have been mapped to CASR Part 91 and the Part 91 MOS as primary sources rather than AIP where possible. If AIP citations are required, the file must be placed in `docs/` and this map updated.
- AC 91-10 (current advisory circular for operations near non-controlled aerodromes; replaces cancelled CAAP 166-01) is cited where applicable. AC 91-10 is published by CASA; currency confirmed per prompt instructions. No numerical page citation — section headings quoted inline where used.

**Forbidden anywhere in the bank:** CAR, CAO, CAAP 166-01, AFM (as a regulatory source), bare "Part 91 MOS" with no chapter, bare "Part 61 MOS" with no schedule/chapter, BMCT, ECT.
